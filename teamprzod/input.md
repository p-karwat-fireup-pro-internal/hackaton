W ramach hackathonu zostanie wyprodukowane MVP mobilnej aplikacji dla e-przeglądów (istniejący wewnętrzny produkt dedykowany zarządcom nieruchomości).
Docelowi użytkownicy: Konserwatorzy pracujący w zarządzanych budynkach.
Główne funkcjonalności: - Lista zadań dla konserwatora (per budynek) - Odhaczanie zadań jako wykonanych + dołączanie zdjęcia jako potwierdzenia - Integracja z centralnym systemem e-przeglądów - Musi działać offline z synchronizacją danych
Zakres: Fokus na aplikacji dla konserwatora (wykonanie w terenie), nie na panelu zarządcy. Specyfikacja zostanie dostarczona osobno.
Backend: Backend istnieje w GitLabie. Do czasu potwierdzenia dostępu do API zakładamy, że backend trzeba będzie zmockować lub częściowo wygenerować.
